Sistema operativo:
Es el software que administra el hardware y permite ejecutar programas. Proporciona el entorno donde funciona el servidor.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador y crear aplicaciones de servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y las respuestas que reciben los usuarios.

Puerto:
Es el punto de comunicación por donde el servidor recibe las solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros de la actividad y los eventos del servidor. Ayudan a detectar errores y monitorear el funcionamiento.
